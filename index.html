Installation Steps

  • Use Chrome or Microsoft Edge browser.
  • Connect the XIAO ESP32-C3 using a USB data cable.
  • Click the Flash Firmware button.
  • Select the correct serial port.
  • Wait for flashing to complete.
  • The board will reboot automatically.

Important Notes

  • Some USB cables are power-only and will not work.
  • Close Arduino IDE or Serial Monitor before flashing.
  • Use a stable USB port directly on your computer.
  • If connection fails, manually enter bootloader mode.

Troubleshooting

  • Board not detected: Install USB serial drivers and reconnect device.
  • Failed to connect: Hold BOOT button while connecting USB.
  • Manifest download failed: Verify manifest.json path and GitHub Pages deployment.
  • 404 firmware error: Confirm BIN files are uploaded correctly.
  • Serial port busy: Close other applications using COM port.

Browser Compatibility

  • Google Chrome (Recommended)
  • Microsoft Edge
  • Opera
  • Chromium-based browsers

Firefox and Safari currently do not fully support Web Serial API.
