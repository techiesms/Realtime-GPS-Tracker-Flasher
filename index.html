Supported Hardware

Board: XIAO ESP32-C3

Chip Family: ESP32-C3

Browser Support: Chrome & Edge

Installation Steps

  1. Connect ESP board using USB
  2. Open Chrome or Edge browser
  3. Click Flash Firmware
  4. Select serial port
  5. Wait for flashing to complete
  6. Device will reboot automatically

Troubleshooting

  • Use data USB cable
  • Close Arduino Serial Monitor
  • Install USB drivers if needed
  • Try another USB port
  • Hold BOOT button while connecting
